Instructions

Click on the coloured squares below to change the colour of MC1/MC2/CC1/CC2.

You can change the colour in 3 ways:

- Select "Grid" or "Spectrum" and tap on the colour to use
- Select "Sliders" to adjust a colour or enter a Hex Colour Code #.

If I have a colour accurate picture of my yarn, I prefer to enter the Hex Colour Code. There are many free online tools that allow you to upload a picture to find the Hex colour code. I use https://imagecolorpicker.com/. Copy the colour code into the sRBG Hex Color # box.

Feel free to take a screenshot of your work to save it.
